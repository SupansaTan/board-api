# Board API

## Installation

```bash
$ npm install
```

## Setting environment file

```bash
$ JWT_SECRET=SECRETsecretSECRET
```

## Running the app

# running on 'localhost:4000'

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Swagger path

```bash
$ localhost:4000/api
```

## Unit tests

```bash
# unit tests
$ npm run test

# unit tests with test coverage
$ npm run test:cov
```
